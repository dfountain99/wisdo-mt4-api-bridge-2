# WISDO v7.0.4 Release Notes

WISDO v7.0.4 combines the heap repair, personalized website recognition, persistent 50% growth celebrations, and crash-safe Culture Lane persistence.

## Highlights

- Bounded queue-only MT4 command state.
- Deterministic copy-command deduplication.
- Critical close authority protected under queue pressure.
- Hard limits across long-lived runtime maps and stalled background flights.
- Animated member name, Culture identity, selected account, balance, equity, floating P/L, and open trades.
- Persistent compact recognition HUD across the member workspace.
- Unique on-screen message at every 50% account-growth level.
- Missed milestones queue in order and persist until acknowledged.
- Culture Lane configuration requires confirmed PostgreSQL persistence before success.
- Leader/receiver membership, profiles, risk settings, symbol policies, Harvest configuration, and linked copier rules survive crashes and redeploys.
- Active Culture Lane relay routes restore automatically after server startup.

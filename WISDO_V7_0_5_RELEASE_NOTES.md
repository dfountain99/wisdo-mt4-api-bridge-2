- Canonical bounded copy-command queue
- Legacy copy-command index migration

- Compact 16 KB Reporter response ceiling
- MT4 poll pacing and burst coalescing

- Copy queue and MT4 response-size health metrics

## Preserved

- PostgreSQL-confirmed Culture Lane saves
- Automatic Culture Lane relay restoration after restart
- Name, rank, account, balance, equity, and floating-P/L recognition animation
- Persistent 50%, 100%, 150%, and subsequent 50% milestone celebrations
- Discord desk creation and recovery
- 77 unique slash commands
- MT4 close authority and immediate high-priority command handling

Recompile and replace the Reporter on every connected terminal. Reporter v1.59 must replace v1.58 on all charts to enable the single-instance polling lease.
